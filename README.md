# 前提

- ターミナル操作ができる

# ハンズオンを進める上で必要なツールの導入
nodeをインストール
https://nodejs.org/ja/download/

以下コマンドを実行してnodeのバージョンを確認し、v12.14.0のようにバージョンが出ればOK（数値はバージョンによって変わる）
```shell script
$ node -v
v12.14.0
```
